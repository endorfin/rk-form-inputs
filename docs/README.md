# Form Inputs Library
## Installation

```
$ npm install @rene.kersten/rk-form-inputs
```

## Usage
```
import Vue from 'vue'
import FormInputs from '@rene.kersten/rk-form-inputs'

Vue.use(FormInputs)
```

## Documentation

#### vuepress dev docs
```
$ npm run docs:dev
``` 

#### vuepress build docs
```
$ npm run docs:build
``` 

## Development

```
$ npm run serve
```
